## Setup

1. Clone the repository
```bash
git clone https://github.com/dragongoose/djs-base-handlerTS.git
cd djs-base-handlerTS
```
2. Change the config to fit your needs

3. Compile the repo
```bash
npx tsc
```

4. Run the bot
```bash
node ./build/index
```
